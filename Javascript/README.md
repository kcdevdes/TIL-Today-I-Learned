# JavaScript Basics

생활코딩 강의를 따라가며 작성한 기초 문법 노트와 작은 HTML 실험 파일을 모았습니다. 한 챕터당 하나의 마크다운 파일과 필요한 경우 예제를 함께 추가했습니다.

## Topics

- `JS01-자바스크립트란?.md` : 언어 소개와 실행 방법
- `JS02 - Number and Character.md` : 숫자/문자 처리, 기본 연산
- `JS03-Variable.md` : 변수 선언과 스코프
- `JS04-Operator.md` : 산술/비교/논리 연산자
- `JS05-IF,ELSE.md` : 조건문과 분기
- `JS06-Repetitive.md`, `JS06_multi_repetitive.html` : 반복문, 중첩 루프
- `JS07-Function.md` : 함수 선언과 인자 전달
- `JS08-Array.md`, `JS09-Dictionary.md` : 배열과 객체 기본 조작

각 챕터와 동일한 이름의 `.html` 파일은 브라우저에서 바로 실행하며 학습한 예제입니다. 필요에 따라 Live Server(혹은 단순 파일 열기)로 확인할 수 있습니다.
