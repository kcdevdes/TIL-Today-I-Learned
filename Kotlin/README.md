# Kotlin Study Notes

튜토리얼을 따라가며 Kotlin 언어 문법과 Android 이전 단계의 기본기를 정리한 공간입니다. Gradle 멀티 모듈 없이 단일 프로젝트 구조를 사용합니다.

## Structure

- `src` : Tutorialspoint를 포함한 여러 문서를 읽으며 작성한 예제 코드
- `notetaking` : 간단한 노트 앱 실습. Jetpack Compose 이전의 XML + ViewBinding 패턴으로 구축했습니다.
- 프로젝트 루트 : 공통 Gradle 설정 및 `build.gradle`, `settings.gradle`
