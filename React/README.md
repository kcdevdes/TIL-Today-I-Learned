# React Practice Hub

벨로퍼트의 React 연재(https://react.vlpt.us/)와 개인 프로젝트를 따라가며 만든 예제들을 모았습니다. CRA와 Vite, TypeScript를 혼합해 사용합니다.

## Structure

- `ch02` : 첫 React 프로젝트 생성 과정 기록
- `ch03-props-and-state` : props/state, 이벤트 핸들링 기초
- `ch05-react-lifecycle-api` : 클래스 컴포넌트 라이프사이클, 에러 경계
- `ch06-react-basics` : 함수형 컴포넌트, Hook 전환
- `ch07-props` : 조건부 렌더링, `useState`, `useRef`, 리스트 렌더링
- `es6` : ES6 문법 리마인드용 샘플
- `phone-book` : CRUD 예제 (검색/필터, 스타일링 포함)
- `mashup-todolist` : 실전 TODO 앱. 상태 관리와 컴포넌트 구조 설계 연습
- `ts-practice` : TypeScript 단일 파일 실험
- `ts-react-tutorial` : React + TypeScript 통합 예제
