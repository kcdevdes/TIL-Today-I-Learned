# Today I Learned (TIL) - Study Note

이 레포지토리는 다양한 언어와 프레임워크를 공부하면서 남긴 코드, 노트, 미니 프로젝트를 한곳에 모아둔 공간입니다. 모든 폴더는 독립적으로 열람할 수 있도록 자체 `README.md`와 간단한 목차를 제공합니다.

이 레포지토리는 제가 2019년 처음 업데이트 이후, 그동안 만들어온 다양한 언어와 프레임워크를 대부분 모아서 만든 집약적 레포지토리입니다. 새로운 공부를 할 때마다 업데이트가 될 예정입니다.

## How to Use

- 원하는 폴더로 이동하면 해당 스택을 공부할 때의 배경, 폴더 구조, 참고 링크를 확인할 수 있습니다.
- 커밋 메시지와 하위 `README.md`를 함께 보면 언제 어떤 챕터/문제를 진행했는지 쉽게 추적할 수 있습니다.
- 학습 메모는 Markdown, 실습은 언어별 표준 툴체인(Gradle, Cargo, npm 등)을 사용했습니다.

필요한 분야부터 자유롭게 살펴보고, 동일한 구조로 새 학습 기록을 추가해 주세요. 즐거운 TIL 여정이 되길 바랍니다.
