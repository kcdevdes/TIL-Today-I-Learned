# C/C++ Practice

C와 C++ 기초 문법, 자료구조, 튜토리얼 실습을 정리한 공간입니다. 호환성을 위해 표준 C11/C++17을 기본으로 사용합니다.

## Structure

- `data-structure/` : 스택, 큐 등 기본 자료구조 구현 실험
- `tutorial/` : 강의 노트를 따라가며 작성한 예제 (`0_강좌 시작전`, `2_참조자`, `4_복사 생성자`, `5_CONST`)
- `today/` : 그날그날 떠오르는 아이디어/문법 테스트
- `seoul/` : 특정 프로젝트(서울시 과제)와 관련된 소규모 코드
- `trash/nextarter-chakra/` : Chakra UI + Next.js 보일러플레이트 참고용

각 디렉터리는 필요한 경우 `main.cpp` 또는 작은 메모 파일을 포함합니다. g++/clang++로 바로 빌드할 수 있도록 단일 소스 구조를 유지합니다.
