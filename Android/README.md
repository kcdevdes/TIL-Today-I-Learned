# Android Playground

Kotlin으로 Android UI 구성, Jetpack 구성요소, 시스템 기능을 연습한 프로젝트 모음입니다. `sample_1`과 `app` 디렉터리는 공통 베이스 앱이며, `chXX_*` 폴더는 특정 강의 챕터와 연결되어 있습니다.

## Structure

- `ch6_view` : 뷰 계층 구조, 이벤트 리스너 기본기
- `ch7_layout` : Linear/Relative/Grid 레이아웃과 Constraint 배치 실험
- `ch8_event` : 터치/제스처 처리, 커스텀 이벤트 콜백
- `ch9_resource` : 문자열/스타일/다국어 리소스 정리
- `ch10_notification` : Notification 채널과 PendingIntent 실습
- `ch11_jetpack` : ViewModel, LiveData 등 Jetpack 컴포넌트 연습
- `sample_1` : 가장 먼저 만든 예제 앱, 위 챕터들의 실험 무대
