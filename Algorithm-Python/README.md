# Algorithm Study with Python

Python 3.12을 사용해 『이것이 취업을 위한 코딩 테스트다』와 BOJ 문제를 풀이한 기록입니다. 각 챕터별로 소스 코드와 간단한 정리 노트를 함께 남기고 있습니다.

## Structure

- `Greedy`  
  - Ch03 예제 풀이와 Baekjoon 필수 문제 모음
  - 탐욕적 선택 조건을 확인할 수 있도록 풀이마다 주석을 정리
- `Implementation`  
  - 시뮬레이션, 문자열 다루기 중심 연습 코드
  - 동일 문제를 다른 방식으로 풀어본 `*-Alt.py` 버전을 함께 두었습니다.
- `DFS-BFS`  
  - 그래프 탐색 템플릿과 Ch05 예제 풀이
  - 재귀 깊이와 큐 사용 예제를 나누어 저장
